Class: PageLoader

Defined in: models/page-loader.coffee
Inherits: EventEmitter

Overview

PageLoader requests a resource located at a given url and emits the response body on success, or an error otherwise.

Events:

pageloaded

  • Emitted with the response body (typically HTML) when the resource has been fetched @type {String}

error

  • Emitted with an Error object on fail @type {Error}

Instance Method Summary

  • # (void) load(options = {}) Loads the page, listeners for pageloaded and error should be added before this method is called
